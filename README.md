<h3><b>Run Locally</b></h3>

* Clone the Repo.
```sh
git clone https://github.com/bibee-emio/Unsplash-bot
```
* Change the Directory.
```sh
cd Unsplash-bot
```
* Install the requirements.
```sh
pip3 install -r requirements.txt
```
* Edit the `config.py` with your own values.
* Run the bot.
```sh
python3 bot.py
```
